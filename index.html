1 Installing the theme

The recommend installation method is by uploading the theme file. If you're unable to use this feature somehow or having issue with that installation method, you can also try to manually copy-paste the source code.

1.1. Installation

  • Log into your Blogger dashboard at https://blogspot.com.
  • Select Theme from the left-hand side of your dashboard page and click the drop-down icon next to the orange Customize button.
  • An popup menu will open, then choose the Restore option.
  • Click the Upload link and browser the XML theme file within the downloaded package.
  • Follow the process and your theme is now active.

1.2. Updating

When updating your theme, follow the installation process as above. In some cases, Blogger will show warning whether to overwrite your existing theme or gadgets. Just choose Yes/OK on this.

We encourage to backup your existing theme before making any changes. By doing backup you restore your old theme anytime if there's any issue during or after the installtion.

To reduce the need for redoing CSS customization, etc, is to use the Add CSS tool within the Theme Designer™. This comes handy especially in adding custom CSS rather than customizing the theme source code directly.

2 Post-installation setup

Now you have the theme installed. The next step is make some adjustment. The following steps will cover instruction for how to set it up properly and get your site running in no time.

2.1. Writing a post

With the post editor you're familiar with, you can upload and attach pictures. All pictures will shrink to fit the container, so we strongly recommend to attach the biggest size possible for the best quality.

In an addition to regular image covers, you can also add a YouTube video. Viral will automatically retrieve video cover from YouTube and generate hi-res images to be used as a post cover.

2.2. Enable post summary

To enable short description on single page, you first need to enable the meta description option. On the Settings page, scroll down to Meta tags section and set the Enable search description to ON.

From here, when you edit or write a new post, you will see a new field named Search Description on the post editor sidebar where you can enter the post short description. It's limited to only 150 characters and is used by search engine to crawl, try to make it more SEO friendly.

2.3. Formatting posts

Viral support multiple post formats, from video to featured post layout. Here's details instruction to help you easily switching between post formats:

  • Regular post without image: Simply write your post as usual without attaching any image or media. This layout will display only the post title on the homepage without a thumbnail and is perfect for writing a journal.
  • Regular post with image: If you attach at least one image to your post, this layout will be used. The default layout features the post title and thumbnail on the homepage.
  • YouTube Video: Make sure that the first media you attach to the post is a YouTube video. You can either manually embed the video or use the text editor to browse the respective video and insert it into your post.
  • Featured post: If you want your post to stand out, add the label "Featured" (case sensitive) and make sure to attach at least one image or a YouTube video. The featured post layout will display the image in a larger format with the title and summary, making the post more prominent.

2.4. Manage Layout and Gadget

The theme comes with various sections that hold gadgets. Each section's label indicates the name of the corresponding gadget, such as Header for the header gadget or Navigation for the Pages gadget. To preserve the theme's layout, most sections are locked and can only hold specific gadgets that match their intended purpose. However, you add any number of gadgets into sidebar and footer.

Add gadgets

To add a gadget, in the left menu, select Layout > Add a Gadget. This will trigger pop-up window that contains all Blogger's default gadgets, including popular posts, contact, label, archive, etc. Those gadgets can be added to your blog by clicking the plus + icon.

Show and hide gadgets

You can control gadget visiblity by clicking on the pencil icon on the gadget. This will open a popup with the visibility option at the top. Uncheck the Visible option to hide the gadget without deleting it. To show it back simply check the option.

2.5. Add social media links

On the Layout page, locate the Social gadget on Footer and click on the pencil icon.

To add an icon enter your social media url and label into the coresponding fields as shown above. Note that in the screenshot above the label has been written in lowercase. This is deliberately done as it will be used as an identifier for the social icons.

At this point, here's the list of social icons you can add on the theme: facebook, twitter, instagram, youtube, behance, tiktok, dribbble, pinterest.

2.6. Customize the theme

Most visual aspect of the theme can be changed through the Blogger’s Theme Designer™. All changes will be reflected live in front of you, without needing to reload the page.

You can access the Theme Designer from your dashboard by navigating to Theme > Customize. To save the changes click the disc icon at the bottom right corner.

2.7. Setting up Logo for Retina Display

If you use an image logo, the image will display at its maximum dimensions. For example, if the image you upload is 200 x 150px, then our theme will display your logo image at 200 x 150px.

To avoid images looking blurry on retina display devices, it is best to use a larger image that is displayed at a smaller size on the website. For example, if you want your logo to show up as a 200 x 150px image on the website, it is best to upload an image that is 400 x 300px, and simply tell the theme to display it at 200 x 150px.

You can upload your logo as usual from the Layout > Header gadget. To update the size of your logo, navigate to Theme Designer™ then choose Advanced. Select Header from the dropdown and resize the logo width by chaging the value of the Logo width option. Logo height will adjust automatically acording to its aspect ratio.

2.8. Creating a contact page

If you have the Contact gadget enabled on the Layout page, you just need to create a page with title Contact to display the form after the content. The Contact gadget will detect the page title, and if it matches the term "Contact" it will automatically display the form.

2.9. Setting up AdSense

Viral provides an effortless way for you to add AdSense to your blog without requiring any coding knowledge. To do this, you just need to visit the Earnings tab and complete a straightforward signup process. If your blog meets the AdSense eligibility criteria, you can easily connect your AdSense account to it.

Once you have successfully synced your AdSense account with your blog, you can start adding ads. To do so, navigate to the Layout tab, where you will see two AdSense sections. Although you can technically add any gadget to these sections, they are specifically designed to accommodate AdSense gadgets only and can host up to one gadget at a time.

Customize the AdSense gadget according to your preferences, and then save your changes. To ensure that the gadget appears correctly on the front-end, remember to check the "Show AdSense" checkbox located at the top before saving.

3 Troubleshoots

Here are some quick steps to fix most theme-related issues. If you experience any issues beyond the scoop this section, please send me a message through https://www.heybi.com/p/get-in-touch.html

3.1. Number of post per page isn't correct

Blogger has introduced a mechanism called "auto pagination" to reduce the loading time of its blogs. This mechanism is to show less entries per page if it detects that the page is heavy to load. Quick workaround to prevent it from working is by adding jump-break/read-more after the first paragraph on each affected post.

Best practice is to compress all images, delete gadgets you don't need, and not doing copy-paste on content.

3.2. Gadgets aren't working properly

If you encounter this issues, try to delete the gadget first and then add it back.

3.3. Error PageList1 gadget (invalid)

It is likely your navigation gadget uses codes or characters not supported with Blogger recent theme HTML updates. To fix it, at the theme Edit HTML page, click Jump to Widget icon. In the dropdown list, selected PageList1, then click Revert gadget to default.

Else you can delete any special and accented character in your navigation label and URL, e.g. #, —, â, î or ô, or use only alphanumeric.

3.4. Related post not working (undefined)

Ensure the reader access is set to Public and each post has at least one label, else the related post script won't be able to read the post data. To update the reader access, navigate to Settings then scroll down to Permissions section. At the Reader access option, choose Public.
